# Convert Base

Convert base number.


## Requires

- [Node.JS](http://nodejs.org/)


## Usage

```javascript
/**
 * Convert base numbers
 *
 * @function: convertBase
 * @params: {String} num | {Number} from | {Number} to
 * @return: {String}
 */

// function(param1 : num, param2 : from, param3 : to)
convertBase('111', 2, 16);
```
